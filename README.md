# ide-D

D ide for Atom

## Features

The features are provided by `serve-d`, `dcd`, `dscanner`, etc.

- Syntax highlightning for D, DML, SDL, and Diet
- Autocompletion
- Datatips
- Linting
- Outline
- Hyperclick
- ...

## Installation

Install from: https://atom.io/packages/ide-d

or

```
apm install ide-d
```
